---

--- 

## Outros problemas

 - Commit com mensagem errada
 - Commit com arquivos errados
 - Merge realizado em branch errado
 - Hitórico de commits não preservado

--- 

## Soluções

 - git reset
 - git merge --no-ff
 - git rebase
 - git pull --rebase
